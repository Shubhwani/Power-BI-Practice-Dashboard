# Power-BI-Practice-Dashboard

📌 Project Overview
This project is a practice dashboard created to strengthen data visualization skills using Power BI. It simulates a product catalog scenario with stock, sales, and pricing data across multiple locations and product groups.

📄 Report Pages
🔹 Report 1 — Overview & KPIs

Key components:
KPI Cards — Quick summary metrics at a glance
Total Stock Value: 115M
Total Stock Balance: 108K
Order Quantity: 75K
Total Sales: 1M
Total Price: 63K
Donut Chart — Order Quantity by Location (Basalt, Apsis, Ceres)
Donut Chart — Price by Product Group
Decomposition Tree — Sum of Stock Balance broken down by Month → Day → Year → Quarter
Date Slicers — Filter by Month, Day, Year, and Quarter

🔹 Report 2 — Detailed Analysis

Key components:
Line Chart — Sum of Price by Brand (Callas, Bellis, Aster) and Product Group
Treemap — Total Price by Product Group (Microwave, Oven, Rice Cooker, Juicer, Mixer)
Waterfall Chart — Sum of Stock Balance by Location (Basalt, Apsis, Ceres, Total)
Slicers — Filter by Product, Brand, Location, and Product Group
